# SohojPaath

**An AI-based assistive application focused on improving reading accessibility for neurodiverse learners.**

## Introduction

SohojPaath is an AI-driven assistive application designed to make reading more accessible for individuals with dyslexia and other neurodiverse learning conditions. Dyslexia affects how the brain processes written language, often causing difficulties in reading, spelling, and sustained focus. SohojPaath addresses these challenges by transforming static text into a dynamic, multi-sensory, and customizable experience. By leveraging Artificial Intelligence (AI) and Machine Learning (ML), the application adapts to each user’s cognitive needs instead of forcing users to adjust to traditional text formats.
